# ⚒️ Process Forge

Process Forge is a Windows desktop utility built with **C#** and **.NET** that helps organize and manage running applications. It provides tools for managing windows, renaming window titles, and launching multiple applications with a single click.

---

## ✨ Features

- 🪟 **Window Management**
  - Show, hide, minimize, maximize, and restore application windows.
  - Easily control multiple running applications.

- ✏️ **Rename Window Titles**
  - Rename application window titles.
  - Makes multiple instances of the same program easier to identify.

- 🚀 **Bulk Application Launcher**
  - Launch multiple applications simultaneously.
  - Save time when setting up your workspace.

- 🔍 **Process-Based Management**
  - Manage applications using their process names.
  - Supports workflows involving multiple instances of the same executable.

- ⚡ **Productivity Focused**
  - Reduce repetitive tasks.
  - Quickly prepare your preferred desktop environment.
